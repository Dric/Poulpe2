Class DbTable

Objet de table de base de données

Les champs de la table sont des objets de type Field, afin de pouvoir gérer leur affichage et leur typage. Cela permet également pour la gestion automatique des enregistrements de table de générer un formulaire possédant automatiquement les restrictions de saisie requises
