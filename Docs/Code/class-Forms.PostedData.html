Class PostedData

Classe de récupération des données envoyées par les formulaires

Cette classe gère également les jetons de sécurité des formulaires

Namespace: Forms
Package: Forms
See: Forms\Form
Located at classes/Forms/PostedData.php

public static array
# get( string $prefix = null, string $dbTablePrefix = null )

Récupère l'envoi d'un formulaire créé via un objet Form et effectue le typage des données retournées

Récupère l'envoi d'un formulaire créé via un objet Form et effectue le typage des données retournées

Les données ne sont pas traitées hors de leur typage, elles ne sont donc pas sécurisées pour un ajout en bdd par exemple.

Les champs sont au format suivant : préfixe_type-de-champ_nom-de-variable(_détails) Les champs de table de bdd sont au format suivant : préfixe-dbTable_nom-de-table_type-de-champ_nom-de-variable(_détails)_id-de-la-ligne Le tableau retourné comporte les noms des variables en clés. Les tableaux ont un sous-index pour indiquer s'il faut sérialiser les valeurs, et un index values pour les valeurs. Les tables de bdd sont dans un tableau nom-de-table => array(ligne => array champs)
