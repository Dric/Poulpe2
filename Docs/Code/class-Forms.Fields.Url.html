Class Url

Champ de saisie d'url (adresse internet)

Ce champ est similaire au type text et a été introduit en html5. Il permet sur les navigateurs qui le supportent de vérifier que la saisie est une url valide

Si Javascript est activé, la validation est effectuée par Poulpe2
