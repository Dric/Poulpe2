Class ACL

Classe de gestion des ACL

Permet de définir des droits sur des composants du site pour des utilisateurs. l'ID d'utilisateur 0 signifie 'tout le monde' et sert à définir des ACL par défaut pour un composant.

Pour gérer les ACL de tout composant, on essaie de faire un tableau associatif de la forme $ACL[$component][$id][$user][$type] = $value Pour les ACL d'un utilisateur, on aura ainsi $userACL[$component][$id][$type] = $value Pour les ACL d'un module, on aura $moduleACL[$user][$type] = $value
